# SalesPilot AI Frontend

React + Vite + TypeScript + Tailwind CSS 前端管理后台。

## 安装依赖

```bash
cd frontend
npm install
```

## 启动开发服务

```bash
npm run dev
```

默认访问：

```text
http://localhost:5173
```

## 打包方式

```bash
npm run build
```

如需修改后端地址，可设置：

```bash
VITE_API_BASE_URL=http://localhost:8000
```
